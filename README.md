# 🥤 汽水走私者 (Soda Smugglers) - 网页版

一款支持3-8人同时在线游玩的策略桌游网页版！

## 🎮 在线游玩

**游戏链接**: https://4fhbn5ro3l3bq.ok.kimi.link

> ⚠️ **注意**: 当前部署版本使用本地模拟服务器。要实现真正的多人在线游玩，请参考 [DEPLOY.md](./DEPLOY.md) 部署完整的服务器版本。

## 📖 游戏背景

lawmakers正在严厉打击汽水，严格的监管催生了利润丰厚的走私活动。每人一瓶是新的法律——因此贿赂、行李箱检查和逮捕都提上了议程。只有一个人能成为汽水大亨！

## 🎯 游戏目标

收集最多的瓶盖，成为汽水走私之王！

## 👥 游戏人数

- 3-8人
- 游戏时长：约20分钟

## 🎲 游戏组件

- **36张行李箱卡**: 0瓶(5张)、1瓶(12张)、2瓶(12张)、3瓶(7张)
- **80个瓶盖代币**: 面值1、3、5、10、20、30
- **7个边境守卫行动令牌**: 接受贿赂、检查行李箱、逮捕
- **8张事件卡**: 为游戏增添变数

## 🎮 游戏流程

### 1. 游戏设置
- 每位玩家获得10个起始瓶盖（两个1、一个3、一个5）
- 玩家轮流担任边境守卫
- 3-4人游戏：每人当2次守卫
- 5-8人游戏：每人当1次守卫

### 2. 每回合流程

#### 阶段1: 揭示事件卡（可选）
边境守卫揭示本回合的事件卡，事件效果持续整回合。

#### 阶段2: 旅行者选择行李
每位旅行者（非边境守卫玩家）获得5张行李箱卡：
- 选择**2张**作为行李（面朝下）
- 选择**1张**作为贿赂（面朝下）
- 弃掉剩余2张

#### 阶段3: 揭示贿赂
所有旅行者同时翻开贿赂卡。

#### 阶段4: 边境守卫行动
边境守卫按顺序使用行动令牌：

**1️⃣ 接受贿赂**
- 接受1名旅行者的贿赂
- 旅行者支付贿赂卡上所示数量的瓶盖给守卫
- 该旅行者被放行，获得行李中的瓶盖

**2️⃣ 检查行李箱**
- 检查1名旅行者的1个行李箱（每位旅行者最多检查1个）
- 被检查的旅行者必须翻开该行李箱

**3️⃣ 逮捕**
- 逮捕1名旅行者
- 该旅行者翻开所有行李
  - **逮捕成功**: 行李中有2+瓶 → 守卫获得这些瓶盖
  - **错误逮捕**: 行李中0-1瓶 → 守卫赔偿2个瓶盖给旅行者

#### 阶段5: 放行
守卫行动结束后，剩余旅行者被放行，获得行李中的瓶盖。

### 3. 游戏结束
所有玩家都担任过边境守卫后，游戏结束。瓶盖最多的玩家获胜！

## 🎴 8张事件卡

| 事件 | 效果 |
|------|------|
| 🌙 **夜班** | 守卫获得1个额外的"接受贿赂"令牌 |
| 🎄 **假日季节** | 每位旅行者打包3个行李箱（但仍只允许携带1瓶） |
| 🍎 **诱惑** | 守卫可以接受所有旅行者的贿赂 |
| 🎂 **生日** | 守卫获得2个瓶盖，本回合可合法携带2瓶 |
| 🐕 **嗅探犬** | 守卫选择1名旅行者，该旅行者展示最多瓶子的行李箱 |
| 👮 **上级警官** | 守卫获得1个额外的"检查"令牌，可重复检查同一人 |
| 📞 **线报** | 守卫获得1个额外的"逮捕"令牌，错误逮捕无惩罚 |
| 💥 **破损** | 守卫失去1个"逮捕"令牌，合法限制变为0瓶 |

## 💡 游戏策略

### 作为旅行者：
- **诚实路线**: 只带1瓶，期待被错误逮捕获得赔偿
- **走私路线**: 带2-3瓶，尝试贿赂守卫
- **心理战**: 用高贿赂引诱守卫，实际行李空空

### 作为边境守卫：
- 观察贿赂金额，判断谁在走私
- 合理利用有限的行动令牌
- 记住已揭示的信息

## 🚀 如何开始游戏

1. 打开游戏链接
2. 创建房间或输入房间代码加入
3. 分享房间代码给朋友
4. 当3-8人都加入后，房主点击"开始游戏"
